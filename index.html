THE BASIC PRINCIPLES

The principle of flexboxes is based on the management of blocks along two axes: either on the main axis (main axis) either on the perpendicular axis (cross axis)


main axis: The direction of the main axis depends on the flex-direction property which we will discuss in detail later.

cross axis: The cross axis being perpendicular to the main axis, it also changes according to the direction assigned to the main axis.
Some properties of the flexbox are to be applied to the parents of the elements to be placed, while others are to be applied to the elements directly

PARENT PROPERTIES

display

It is used to identify the parent of the elements to be placed, it usually takes the value flex, but there is also inline-flex.

flex-direction: defines the direction of the main axis

  • row: from left to right
  • row-reverse: from right to left
  • column: from top to bottom
  • column-reverse: from bottom to top

flex-wrap: used to identify the parent of the elements to be placed, it usually takes the value flex, but there is also inline-flex.

  • nowrap: everything is on the same line
  • wrap: items wrap when they run out of space, top to bottom
  • wrap-reverse: items wrap when they run out of space, from bottom to top

flex-flow

It allows to condense the writing of the flex-direction and the flex-wrap. Example : flex-flow: column wrap

justify-content: defines the alignment along the main axis

  • flex-start: lines up the items from the start of the container
  • flex-end: lines up the items from the end of the container
  • start: lines up the items from the start of the writing-mode direction
  • end: lines up the items from the end of the writing-mode direction
  • center: lines up the items in the center, along the line
  • space-between: distributes the items evenly, from the first to the last line
  • space-around: distributes the items evenly, with the same space on both sides of each item. This means that the first and last one will have on their outer side half of the space there is between their other side and the next item.
  • space-evenly: distributes the items to let them be separated by the same space, even on outer sides

align-items

It defines the alignment of the items along the cross axis

  • flex-start: lines up the items from the start of the container
  • flex-end: lines up the items from the end of the container
  • center: lines up the items in the center
  • baseline: lines up the items following the text baseline
  • stretch: adjust the size of the item to fill the available space

align-content

When there is extra space along the cross axis, it aligns the different lines of a wrap or wrap-reverse flexbox

PROPERTIES OF CHILDREN

order

It defines the order in which elements appear in the flex parent

flex-basis

It sets the initial main size of a flex item. It sets the size of the content box unless it is set with box-sizing

flex-grow

It sets the grow factor of a flex item (it takes more space / ratio)

flex-shrink

This is the same thing as flex-grow but it shrinks the size of the item

align-self: It allows an item to be placed independently along the cross axis.

  • auto: takes the parent's value
  • flex-start: lines up the items from the start of the container
  • flex-end: lines up the items from the end of the container
  • center: lines up the items in the center
  • baseline: lines up the items following the text baseline
  • stretch: adjust the size of the item to fill the available space
